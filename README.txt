How to use:
1. Open index.html in a browser.
2. Public pages are available through the left navigation.
3. Admin dashboard is available at the Admin link.
4. Default admin login:
   username: admin
   password: projectadmin
5. All admin changes are stored in browser localStorage and instantly update the public UI.

Included features:
- Homepage with hero, sports overview, tournament progress, and recent updates
- Large overall leaderboard
- Separate sport pages with standings, status, notes, points, rank, and leaderboard points awarded
- Not Started Yet card when no sport data exists
- Match results page
- Performance charts
- Small admin panel for houses, sports, match results, standings, leaderboard view, and announcements
